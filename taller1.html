SQL Querys

1- Alias/Sum/count

SELECT avg(edad) as Promedio_edad_jugadores, count(*) Numero_Jugadores FROM jugadores

2- Group by/Orden by/Where

- SELECT Nombre_de_equipo, count(*) as EQUIPOS from equipos where Nombre_de_equipo='Milan' group by Nombre_de_equipo

- SELECT Nombre, Apellido, Edad FROM jugadores ORDER BY Edad

3- Relaciones entre Claves Primarias / Where

- SELECT * from Partidos P, Equipos E where E.id_partidos = P.id_partidos

- SELECT P.fechas, E.Nombre_de_equipo from Partidos P, Equipos E where E.id_equipos = P.id_partidos

4- Inner/Outer Join/On

- SELECT J.Nombre, J.Apellido, E.Nombre_de_equipo from Jugadores J INNER JOIN Equipos E ON E.id_equipos = J.id_equipos

- SELECT J.Apellido, J.Edad, E.Nombre_de_equipo from Jugadores J LEFT OUTER JOIN Equipos E ON E.id_equipos = J.id_equipos
